# Changelog

All notable changes to FindDrive will be documented in this file.

Format: `[YY.MM.NN]` — рік.місяць.порядковий_номер_релізу_за_день

---

## [24.06.30] — 2026-06-25

### Fixed
- Firestore rules: `bookings` тепер відкриті для читання — `getBusySlots` більше не блокується
- `_bookSlot`: inline-помилка замість `alert`, перевірка авторизації з повідомленням
- При `permission-denied` показується підказка оновити Firestore Rules

---

## [24.06.28] — 2026-06-25

### Added
- PWA: `manifest.json` з іконками 32/192/512px, standalone-режим, тема #1a7a4a
- PWA: `sw.js` — service worker з precache (index.html + іконки) та stale-while-revalidate
- Мета-теги: theme-color, apple-mobile-web-app, description
- Кнопка "Додати на екран" з'являється автоматично в Chrome/Safari

---

## [24.06.29] — 2026-06-25

### Fixed
- Анкета інструктора: якщо email вже зареєстрований — автоматичний вхід замість помилки
- Якщо auth/email недоступний — анкета все одно надсилається (без прив'язки акаунту)
- Конкретні повідомлення про помилки в формі замість generic alert
- Firestore rules: create instructor дозволено без авторизації (статус pending)

## [24.06.27] — 2026-06-25

## [24.06.26] — 2026-06-25
