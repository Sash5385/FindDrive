# Changelog

All notable changes to FindDrive will be documented in this file.

Format: `[MAJOR.MINOR.PATCH] — YYYY-MM-DD`

## [0.3.0] — 2026-06-24

### Added
- Google Maps JavaScript API замість placeholder SVG-карти
- Кастомні маркери для 3 демо-інструкторів (кольори: зелений, рожевий, синій)
- InfoWindow при кліку на маркер: ім'я, рейтинг, ціна, кнопка "Записатись"
- Кнопка "Моє місце" в хедері секції — геолокація через `navigator.geolocation`
- Кастомні стилі карти в палітрі дизайн-системи (#f0f2f0 фон, зелені дороги)
- Карта: висота 420px (мобільна 320px)

### Changed
- Секція карти: прибрано overlay CTA, кнопка переміщена в section-top

## [0.1.0] — 2026-06-24

### Added
- Початковий статичний лендінг `index.html`
- Навбар з логотипом, посиланнями, кнопками "Увійти" та "Стати інструктором"
- Hero-секція з пошуком по локації, кнопкою геолокації, популярними районами
- Секція карток інструкторів (сітка 3 колонки, аватар, рейтинг, відстань, теги, ціна)
- Секція "Як це працює" — 4 кроки в кольорових картках
- Placeholder-карта з піна-мітками інструкторів
- Footer з посиланнями та копірайтом
- `CNAME` — кастомний домен `finddrive.id4drive.pro`
